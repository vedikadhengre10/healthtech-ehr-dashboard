# 🏥 PulseCore EHR & Telemetry Dashboard

A modern, responsive, and interactive Electronic Health Record (EHR) and Clinical Telemetry Management System built using HTML, CSS, and JavaScript.

PulseCore simulates a real-world hospital monitoring environment with patient management, live telemetry tracking, clinical dashboards, alert monitoring, accessibility controls, and admission workflows.

---

## ✨ Features

### 📊 Clinical Dashboard
- Real-time hospital overview
- Bed occupancy tracking
- Staff availability metrics
- Discharge queue monitoring
- Critical alert statistics

### 👨‍⚕️ Patient Management
- Patient directory
- Detailed patient profiles
- Medical history records
- Prescription management
- Encounter timelines
- Laboratory reports

### ❤️ Live Telemetry Monitoring
- Real-time vital signs simulation
- Heart Rate (HR)
- Blood Pressure (BP)
- Oxygen Saturation (SpO₂)
- Temperature monitoring
- Critical alert generation

### 🚨 Alert Management System
- Active telemetry alerts
- Critical patient notifications
- Alarm prioritization
- Alert dismissal controls
- Audio alarm support

### 🏥 Admission Workflow
- Multi-step patient admission wizard
- New patient registration
- Room assignment
- Department allocation
- Clinical record creation

### ♿ Accessibility Features
- Dark Mode
- Light Mode
- High Contrast Mode
- Adjustable font sizes
- Dyslexia-friendly font mode
- Audio accessibility controls

### 🔍 Advanced Search & Filtering
- Global patient search
- Triage queue filtering
- Department filtering
- Severity-based sorting

### 📱 Responsive Design
- Desktop optimized layout
- Tablet support
- Modern glassmorphism UI
- Mobile-friendly components
